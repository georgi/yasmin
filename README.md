Yasmin
======

A simple c-like language for my own pleasure.

## Types

* bool
* byte
* int16
* int32
* int (defaults to int64)
* float
* void
* pointer (like `byte*`)
* function

## Functions

C-style types and function declaration.

```
int add(int x) {
  x + x;
}

add(10);
```

## Variables

Variables can be assigned only once.

```
int x = "Hello" in
int y = " World" in
puts(x + y);
```
